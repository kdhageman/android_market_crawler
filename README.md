# PyStoreCrawler
A crawler that retrieves Android application meta-data and APK files from a selection of markets.

#### Supported markets
A diverse set of markets is being crawled, both from the west and China. 

##### Western
- SlideME 
- F-Droid
- APKMirror
- APKMonk

##### Chinese
- Tencent
- Xiaomi Mi
- 360
- Baidu
- Huawei
